# Final_Deliverable_Applied_AI_Jin
Steps for running the code:
1. Please download the dataset from https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
2. Please add a 'Data' folder to the same directory as the code
3. Please place the downloaded data to the 'Data' folder
4. Run the code 

P.S. Note that the code shown on github did not include two hyperparameter tuning result plots. Please view them as png files in the same directory as the source code on github repository.
